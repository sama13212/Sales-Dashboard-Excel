# 📊 Sales Dashboard (Excel)

## Project Overview

This project presents an interactive Excel dashboard that analyzes sales performance across product categories, regions, customer demographics, and payment methods. The dashboard enables users to monitor key business metrics and supports data-driven decision-making.

---

## Tools & Features Used

- Microsoft Excel
- Power Query (Data Cleaning)
- Data Model (Relationships)
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dashboard Design

---

## Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Returned Orders
- Average Delivery Days

---

## Business Insights

- Electronics is the highest-performing category, contributing nearly half of total sales.
- Beauty products generate the lowest sales.
- South region records the highest profitability.
- East region has the longest delivery time.
- Customers aged 56–70 represent the largest customer segment.
- Credit cards are the most preferred payment method.
- Monthly sales decline significantly in September.
- Return rate remains low at approximately 5.5%.

---

## Recommendations

- Increase investment in Electronics.
- Improve marketing performance for Beauty products.
- Optimize logistics in the East region.
- Investigate Central region's lower profitability.
- Expand loyalty programs for older customers.
- Encourage digital wallet adoption.
- Analyze September's sales decline.
- Continue monitoring return reasons.
